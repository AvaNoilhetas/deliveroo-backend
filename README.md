## Installation

1. Clone the repo

2. Install NPM packages

   ```sh
   yarn
   ```

3. Create a .env file with :

   ```JS
    PORT=3000
    FRONT_WEB_SITE=YOUR_URL
   ```

4. Run the project

   ```JS
   npx nodemon index.js
   ```
